How to add Puppet syntax highlighting to Dokuwiki

Fri 05 July 2013
By frommelmak
Category: English
Tags: puppet dokuwiki

As you probably know, Dokuwiki relies on GeSHI for syntax highlighting. Unfortunately, the current version of GeSHI does not support Puppet language. Nevertheless GeSHI allows you to add new languages easily by creating new language files, so to add Puppet support you just need to get a languaje file for Puppet and copy it into the inc/geshi folder.
